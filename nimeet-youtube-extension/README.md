# NIMeet YouTube Broadcast Extension

Расширение для Chrome, которое добавляет кнопку "Транслировать в NIMeet" на YouTube.

## Установка

### Для разработчиков (Developer Mode)

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `nimeet-youtube-extension`
5. Расширение установлено! ✅

### Для пользователей

Скачайте расширение по ссылке из NIMeet (кнопка скачивания рядом с YouTube).

## Использование

1. Присоединитесь к созвону в NIMeet
2. Откройте YouTube в новой вкладке
3. Найдите видео, которое хотите транслировать
4. Нажмите кнопку **"NIMeet"** в правой части видеоплеера
5. Видео начнёт воспроизводиться у всех участников созвона!

## Как это работает

```
[YouTube]  ──▶  [Extension]  ──▶  [NIMeet]
  кнопка      передаёт URL       плеер
```

- Расширение определяет, находитесь ли вы в созвоне NIMeet
- Если да — кнопка на YouTube становится активной (зелёной)
- При нажатии URL видео передаётся в NIMeet
- NIMeet автоматически запускает YouTube плеер

## Файлы

```
│   └── nimeet.js           # Скрипт для NIMeet (мост)

│   └── service-worker.js   # Фоновый сервис
└── styles/
    └── youtube-button.css  # Стили кнопки на YouTube
```

## Разработка

### Тестирование изменений

1. Внесите изменения в код
2. Перейдите на `chrome://extensions/`
3. Нажмите кнопку обновления на карточке расширения
4. Обновите страницы YouTube и NIMeet

### Логи

Откройте DevTools (F12) и смотрите консоль:
- На YouTube: `[NIMeet Extension] ...`
- На NIMeet: `[NIMeet] ...`
